# Photo Reference Checklist — The Miners 1928

Every photo spot on the website has a short code (e.g. `WED-3`). Use the
code to tell us exactly which spot a photo is for.

**The codes are shown on every page right now** as small numbered tags over
each photo spot. (To hide them for a clean preview, add `?photos=off` to the
address, e.g. `weddings.html?photos=off`.)

**Legend**

- 🔴 **have** — a photo is in place.
- 🔴 **replace** — a stand-in photo is there now; a better one is wanted.
- 🔵 **needs photo** — empty slot; a brand-new photo is needed.

**Totals:** 63 photo spots — 44 have, 9 to replace, 10 empty slots needing a photo.

_Generated by `node photo-spots.js`. Re-run after adding or removing images._
